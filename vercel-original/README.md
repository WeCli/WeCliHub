# Legacy ClawcrossHub (Flask)

This folder contains the original single-file Flask implementation extracted from `ClawcrossHub 2.zip`.

## Deploy on Vercel for side-by-side comparison

1. Create a **new** Vercel project from the same GitHub repo.
2. Set **Root Directory** to `vercel-original`.
3. Keep Framework Preset as `Other` (or let Vercel auto-detect Python).
4. Deploy.

This does not modify the main Next.js deployment at repo root.
